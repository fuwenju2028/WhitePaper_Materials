# Glossary of Key Terms

## Structural Irrational Numbers / Endogenously Structural Expanding Transcendentals

| 中文                   | English                                          | Abbreviation | Definition                                                   |
| ---------------------- | ------------------------------------------------ | ------------ | ------------------------------------------------------------ |
| 结构无理数             | Structural Irrational Number                     | SIN          | A class of irrational numbers whose decimal expansions, while passing conventional randomness tests, exhibit measurable, non-random, multi-scale deterministic patterns |
| 内源性结构展开型超越数 | Endogenously Structural Expanding Transcendental | ESET         | A refined nomenclature for SIN, emphasizing origin (endogenous), form (structural), mode of manifestation (expanding), and mathematical category (transcendental) |
| 内源性                 | Endogenous                                       | —            | Structure originates from the mathematical definition itself, rather than being externally imposed or artificially extracted |
| 展开型                 | Expanding                                        | —            | Structure reveals itself layer by layer upon expansion; expansion length correlates with observable structure depth |

## Six Core Properties

| 中文       | English               | Definition                                                   | Key Indicator                                         |
| ---------- | --------------------- | ------------------------------------------------------------ | ----------------------------------------------------- |
| 过程性     | Processuality         | Long-range memory in the sequence; current state influenced by historical states | Hurst exponent H (H > 0.5 for persistence)            |
| 受控随机性 | Controlled Randomness | Passes classical randomness tests but systematic deviations in higher-order statistics | Passes χ²/runs tests; non-flat power spectrum         |
| 包容性     | Inclusiveness         | All finite-length patterns appear within a finite prefix     | Coverage rate C(L,N); first occurrence positions      |
| 交替性     | Alternation           | Dynamic balance of transitions between adjacent digits       | Transition symmetry error; odd-even transition ratio  |
| 层级性     | Hierarchy             | Statistical properties scale with observation scale according to a specific law | Scaling exponent H_h (π: ≈ -0.009; white noise: -0.5) |
| 嵌套性     | Nesting               | Recursive containment relationships among periodic patterns at different scales | Nesting degree Nest(S); average nesting depth         |

## Statistical Concepts

| 中文       | English                  | Definition                                                   |
| ---------- | ------------------------ | ------------------------------------------------------------ |
| Hurst指数  | Hurst Exponent           | H = 0.5 for white noise (no memory); H > 0.5 indicates persistence (long memory); H < 0.5 indicates anti-persistence |
| 标度指数   | Scaling Exponent         | Slope in log-log plot of fluctuation vs. scale; characterizes how statistical properties change with scale |
| 正规数     | Normal Number            | A real number whose decimal expansion has all digits and all finite patterns appearing with equal asymptotic frequency |
| 超越数     | Transcendental Number    | A real or complex number that is not algebraic (not a root of any non-zero polynomial with integer coefficients) |
| 正规数猜想 | Normal Number Conjecture | The conjecture that π, e, and other fundamental constants are normal numbers |
| 替代数据法 | Surrogate Data Method    | A statistical testing method that randomizes the phase while preserving the amplitude spectrum |

## Philosophical Concepts

| 中文         | English                       | Definition                                                   |
| ------------ | ----------------------------- | ------------------------------------------------------------ |
| 耦洽自解     | Coupled Self-interpretation   | A property conjectured for structural irrational numbers: the generating rules (coupling) and the information to reverse-engineer those rules from the sequence (self-interpretation) are deeply connected |
| 比例随机演化 | Proportional Random Evolution | A conjecture concerning the proportional relationship between structural unit scale (L) and its first significant occurrence position (N): L ∝ ρ(N)·N^α, where ρ(N) is a slowly randomly evolving factor |
| 结构深渊     | Structural Abyss              | A metaphor for the nesting property: within seemingly random digits, one can recursively discover deeper and deeper structural layers |
| 随机面纱     | Random Veil                   | The phenomenon by which a structural sequence passes all classical randomness tests, masking its underlying order |
| 结构内核     | Structural Core               | The underlying deterministic patterns revealed by higher-order statistical analyses |

## Usage Notes for Academic Writing

1. **First use of a key term** should include the English translation in parentheses. Example: "过程性 (Processuality)"

2. **Abbreviations** should be defined at first use. Example: "Structural Irrational Number (SIN)"

3. **For international journals**: Use English terms throughout; Chinese equivalents can be omitted or placed in footnotes.

4. **For Chinese journals with English abstracts**: Use Chinese terms in the main text and provide the English equivalents in the abstract and a glossary.

5. **Consistency**: Once a translation choice is made, use it consistently throughout the document.
